Step by step Installing Oracle 12c on CentOS 7 and Configuring with Screenshots

First reffer Part1.pdf 
It contains steps to configure before insalling the Oracle 12c.
Neccessary steps to follow in part 1:

	Need to install dependencies for RDBMS
	Need to Create Oracle user account and 2 groups named "oinstall"  and "dba".
	Add and apply required kernel parameters to /etc/sysctl.conf
	Edit limit.conf file “vim /etc/security/limits.conf”
	Create necessary directories such as u01, u02
	Assign necessary permissions for created directories
